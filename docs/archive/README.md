# Desktop Archive

This directory holds dated desktop task artifacts that are useful for historical reference but should not keep
cluttering the side-repo root.

Current archived artifacts:
- `agent_prompt.md`
- `detailed.md`
- `i18n-fix-report.md`
- `qa-report.md`

Rule:
- keep durable workflow and product docs in `apps/desktop/README.md` or `apps/desktop/docs/`
- move dated runbooks, QA transcripts, and generated-style audit notes here
- leave local one-off artifacts untracked unless there is an explicit reason to preserve them in git
